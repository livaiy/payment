# buatkan-website-kursus-online

Buatkan website kursus online fullstack dengan fitur:
- Landing page modern dan responsif
- Sistem login & register (student dan instructor)
- Dashboard instructor untuk upload video dan materi
- Dashboard student untuk melihat kursus yang dibeli
- Sistem pembayaran menggunakan Xendit
- Notifikasi setelah pembayaran berhasil
- Desain minimalis dan profesional


## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the URL shown in the terminal.
